O que é o DRCA?

Para mais informações:

O DRCA é um site para gerenciamento Acadêmico, onde é possível listas as Secretarias, os Departamentos e os Alunos registrados. Além disso, é possível editar as Secretarias e gerar informações dos Alunos em Excel e PDF.

O site é desenvolvido em Python utilizando o Framework Django e é possível encontrar o setup de instalação e o arquivo do projeto para download neste link
